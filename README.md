
# VetCloud 🐾

VetCloud es una plataforma web diseñada para optimizar la administración de veterinarias, facilitando el control de inventarios, historias clínicas y la gestión de pacientes a través de una interfaz intuitiva y moderna.


## 🚀 Características Principales

- Dashboard Administrativo: Vista general del estado de la clínica en tiempo real.
- Gestión de Pacientes: Registro detallado de mascotas, incluyendo historial clínico y datos del propietario.
- Control de Inventario: Manejo de stock de insumos médicos y productos de venta.
- Citas y Consultas: Calendario para la programación de servicios médicos.
- Interfaz Adaptativa: Diseño responsivo para su uso en computadoras y dispositivos móviles.
